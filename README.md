# 基于协同过滤算法的Python图书推荐系统 python333

## 项目概述

本项目是一个基于协同过滤算法的Python图书推荐系统，采用前后端分离的技术架构，后端使用Python Flask框架，前端使用Vue.js，数据库采用MySQL。

## 技术栈

- 后端：Python Flask
- 前端：Vue.js
- 数据库：MySQL

## 运行环境

- Python 3.12
- MySQL 8.0

## 功能模块

### 用户端功能

- 用户注册登录：支持账户注册、安全登录和身份认证
- 个性化推荐：基于协同过滤算法实现智能图书推荐
- 图书浏览：展示图书详情、封面、简介和评分
- 搜索功能：支持按书名、作者和分类搜索图书
- 评分评论：用户可以对图书进行评分和发表评论
- 图书论坛：分享阅读感悟，支持回帖交流
- 个人中心：管理个人信息和收藏图书

### 管理端功能

- 用户管理：管理用户账户和权限
- 图书管理：添加、编辑和删除图书信息
- 分类管理：维护图书分类体系
- 数据统计：查看系统运营数据
- 系统配置：设置系统参数和配置
- 内容审核：管理评论和论坛内容

## 系统角色

- 普通用户：使用系统进行图书浏览、搜索、评分、评论等操作
- 管理员：负责用户管理、图书管理、内容审核等功能
